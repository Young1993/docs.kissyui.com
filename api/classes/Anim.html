Parameters:

  • elem String | HTMLElement | KISSY.Node | Window | 普通的Object

    作用动画的元素节点或窗口（窗口时仅支持 scrollTop/Left）

  • props Object

    动画结束的 dom 样式值

  • [duration=1] Number optional

    默认为 1 , 动画持续时间, 以秒为单位

  • [easing="easeNone"] String optional

    默认为 "easeNone", 动画平滑函数, 可取值 "swing", "easeNone", "linear", "easeIn", "easeOut", "easeBoth","easeInStrong", "easeOutStrong","easeBothStrong","elasticIn","elasticOut", "elasticBoth","backIn","backOut","backBoth", "bounceIn","bounceOut","bounceBoth", "cubic-bezier(p1x, p1y, p2x, p2y)（所有取值必须在[0,1]之间）". 效果预览, 可以参考easing可视化和cubic-bezier可视化

  • [completeFn] Function optional

    动画到最后一帧后的回调函数

    Note: 构造器也可以如下面这样传参

    Anim (elem, props[, config]) 得到绑定于某个 dom 节点的动画实例

  • elem String | HTMLElement | KISSY.Node | Window | 普通的Object

    作用动画的元素节点或窗口（窗口时仅支持 scrollTop/Left）

  • props Object

    动画结束的 dom 样式值

  • [config] Object optional

    动画配置项

    • [duration=1] Number optional

      单位秒。默认 1 秒.动画持续时间

    • [easing="easeNone"] String optional

      默认 ‘easeNone’. 动画平滑函数

    • [queue=undefined] String | False | Undefined optional

      所属队列名称. 默认undefined. 属于系统内置队列, 设置 false 则表示该动画不排队立即执行

    • [complete] Function optional

      动画到最后一帧后的回调函数

    • [useTransition=false] Boolean optional

      是否使用css3 transition提升性能。 在 useTransition 为 true 的时候，easing 的值必须是 w3c 规定的时间函数名称。 具体为： ease | linear | ease-in | ease-out | ease-in-out | cubic-bezier(p1x, p1y, p2x, p2y)

    默认为false. false 时, 动画会在当前帧直接停止（不触发 complete 回调）. 为 true 时, 动画停止时会立刻跳到最后一帧（触发 complete 回调）
