Constructor

Control

(
  • config
)

Defined in src\api\component\control.js:7

Parameters:

  • config Object

    配置项，有如下配置：

    • [content] String optional

      可选, 设置内容 html

    • [disabled] Boolean optional

      可选，该组件是否初始禁用

    • [elAttrs] Object optional

      可选，附加给组件根节点的属性键值对

    • [elBefore] KISSY.Node optional

      可选，组件根节点的渲染到该节点之前

    • [elCls] String optional

      可选，附加给组件根节点的样式类

    • [elStyle] Object optional

      可选，附加给组件根节点的内联样式

    • [focused] Boolean optional

      可选，该组件是否初始获得焦点

    • [height] optional

      可选，组件的高度，单位像素

    • [parent] Control optional

      可选，该组件的父组件

    • [prefixCls] String optional

      可选，默认 “ks-” . 组件的 css 样式类前缀 . 例如假设组件为 menu ，则该组件内的样式类名为 {prefixCls}menu，默认为 “ks-menu”.可用于实现自定义皮肤

    • [prefixXClass] String optional

      可选，组件 prefix 的超类。只在config中使用。当超类未被指定时，用这个做超类

    • [render] KISSY.Node optional

      组件要应用的节点。默认 S.all(“body”)，组件根节点的渲染为该节点最后一个节点

    • [srcNode] KISSY.Node optional

      可选，组件从页面中已存在的该节点中渲染而来.srcNode 时设置其他属性不起作用，属性通通在 html 标签中指定，并且 html 标签必须包含完整结构，例如 content 节点必须存在

    • [visible=true] Prefix optional

      默认 true ，是否显示.只是为组件的根节点添加/删除 {component}-hidden 形式的 css class，自行指定具体的 css 样式

    • [width] Number optional

      可选，组件的宽度，单位像素

    • [x] Number optional

      横轴位置

    • [y] Number optional

      纵轴位置

    • [xy] Number[] optional

      横纵坐标

    • [zIndex] Number optional

      z-index 值
